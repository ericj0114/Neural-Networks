HOW TO RUN

Make sure JupyterHub is installed, either on local machine or remote server with GPU

Pull segmentation.ipynb file from repository

Download 'segmentation_data' folder 

Make sure .ipynb file and 'segmentation_data' folder exist in the same directory

Open segmentation.ipynb in JupyterHub and run to reproduce results

*NOTE: make sure ALL packages imported in the header are installed in Python enviroment
