HOW TO RUN

Make sure JupyterHub is installed, either on local machine or remote server with GPU

Pull multi_wbc.ipynb file from repository

Download 'cell' folder 

Make sure .ipynb file and 'cell' folder exist in the same directory

Open multi_wbc.ipynb in JupyterHub and run to reproduce results

*NOTE: make sure ALL packages imported in the header are installed in Python enviroment
